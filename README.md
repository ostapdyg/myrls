## Team

## Descripiton

Written in C

## Prerequisites

 - **C++ compiler** - needs to support **C++17** standard
 - **CMake** 3.15+
 

## Usage

```bash
myrls [FILE]
```

If FILE is not provided, it is interpreted as current directory

Help flags `-h`/`--help` support is available.
